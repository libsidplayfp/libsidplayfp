Detailed Description

Commodore 64 emulation core.

It consists of the following chips:

  • CPU 6510
  • VIC-II 6567/6569/6572/6573
  • CIA 6526/8521
  • SID 6581/8580
  • PLA 7700/82S100
  • Color RAM 2114
  • System RAM 4164-20/50464-150
  • Character ROM 2332
  • Basic ROM 2364
  • Kernal ROM 2364
