Detailed Description

8580 integrator 

              +---C---+
              |       |
vi -----Rfc---o--[A>--o-- vo
              vx

IRfc + ICr = 0
IRfc + C*(vc - vc0)/dt = 0
dt/C*(IRfc) + vc - vc0 = 0
vc = vc0 - n*(IRfc(vi,vx))
vc = vc0 - n*(IRfc(vi,g(vc)))

IRfc = K*W/L*(Vgst^2 - Vgdt^2) = n*((Vddt - vx)^2 - (Vddt - vi)^2)

Rfc gate voltage is generated by an OP Amp and depends on chip temperature.
